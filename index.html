MCube Lab for Autonomous Robotic Manipulation

Computer Vision Researcher

  • At MCube Lab, my project focuses on integrating computer vision to improve high resolution tactile sensing to automate and accurately perform robotic manipulation tasks. Using my project, the robot is able to accurately isolate an object from its surroundings and reconstruct 3D point clouds of the object given RGB-D images using a next-best pose estimation model. I have also convolutional-neural nets (CNN) to both evaluate generated models and also to map reconstructed 3D object models with manipulated objects for pose estimation. To further improve point cloud models, I have also implemented Depth Image Completion to extrapolate unknown depth values in the dataset using pytorch.

    Main Tools Used: Python, GQ-CNN, Pyrender, OSMesa, Meshlab, Pytorch

SmartAlarm

Personal Project

Many people struggle with waking up in the morning even with an alarm. The temptation to just hit the snooze button and go back to sleep is strong. Modern day alarm apps, like the iOS clock app, do not provide an incentive to actually get out of bed. We have developed a solution to this problem. SmartAlarm is an IoT device that incentivizes students to wake up and get out of bed when their alarm rings. Alarms can either be set on the user’s account on our web app, or can be set through voice commands, that are collected from a microphone and parsed using the Google Speech API. We customized alarms by setting different frequencies on the buzzer to generate desired notes.Once the alarm rings, the user can either perform 10 jumping jacks or win a game of cup pong to shut off the alarm. We calibrated jumping jacks to each user’s personal thresholds such that the device accurately and consistently marks jumping jacks, but not jumps or simple movements of the arm (in other words, you can’t fool SmartAlarm). A game of cup pong is demonstrated in the video, where distance of the ball is calculated using accelerometer and gyroscope data. User set alarms and reminders are curated and displayed on our web app.

Hardware Parts: ESP32 Arduino, Gyroscope, Accelerometer, Arduino buzzer
Software Tools: C++, Python - Pandas, HTML, CSS, JS, Google Speech API

Juarez Labs - Illinois Institute of Technology

omputational Biology Intern

  • At Juarez Labs, I worked on a novel approach to finding more effective antibiotics against C. Trachomatis. I used the Chimera graphical modeller to create atomic-resolution models for various human and bacterial proteins to predict and simulate drug effectiveness. Because models of these proteins don’t currently exist, I generated these models after selecting template proteins from the BLAST program based on similarities in amino acid sequences and residual graphs. I developed these models using python and the Chimera graphical modeller and tested my simulations on infected HeLa cells.
